# Seedream3.0 - 文本生图 API (Text to Image)

## 接口概述

由 Seedream3.0 驱动的图像生成 API。

提交任务后，您可以使用统一的查询接口检查进度并获取结果。为了在生产环境中获得更好的体验，强烈建议使用 `callBackUrl` 参数来接收生成完成的自动通知，而不是轮询状态接口。

---

## 基础信息

* **基础路径 (Base URL)**: `https://api.kie.ai` (正式环境)
* **请求路径**: `/api/v1/jobs/createTask`
* **请求方法**: `POST`
* **Content-Type**: `application/json`

### 身份验证 (Authentication)

所有 API 都需要通过 Bearer Token 进行身份验证。

* **获取 API Key**: 请访问 [API Key 管理页面](https://kie.ai/api-key) 获取。
* **使用方法**: 在请求头 (Headers) 中添加：
    `Authorization: Bearer YOUR_API_KEY`
* **注意事项**: 请妥善保管您的 API Key，切勿泄露给他人。若怀疑 API Key 泄露，请立即在管理页面重置。

---

## 请求参数 (Request Body)

| 参数名 | 类型 | 必填 | 默认值 | 描述 |
| :--- | :--- | :---: | :--- | :--- |
| **`model`** | String | 是 | `bytedance/seedream` | 用于生成的模型名称。必须为 `bytedance/seedream`。 |
| **`callBackUrl`** | String | 否 | - | 接收生成任务完成更新的 URL（推荐生产环境使用）。系统会在生成完成时通过 POST 请求将状态和结果发送至此 URL。 |
| **`input`** | Object | 否 | - | 生成任务的输入参数对象（详见下方 `input` 对象说明）。 |

### `input` 对象说明

| 参数名 | 类型 | 必填 | 默认值 | 描述 |
| :--- | :--- | :---: | :--- | :--- |
| **`prompt`** | String | 是 | - | 用于生成图像的文本提示词。（最大长度：5000 字符） |
| **`image_size`** | String | 否 | `square_hd` | 图片尺寸。可选值：`square`, `square_hd`, `portrait_4_3`, `portrait_16_9`, `landscape_4_3`, `landscape_16_9` |
| **`guidance_scale`** | Number | 否 | `2.5` | 控制输出图像与输入提示的匹配程度。值越高，提示词相关性越强。(范围：1 - 10，步长：0.1) |
| **`seed`** | Integer| 否 | - | 随机种子，用于控制图像生成的随机性。 |

## 请求示例

```json
{
  "model": "bytedance/seedream",
  "callBackUrl": "[https://your-domain.com/api/callback](https://your-domain.com/api/callback)",
  "input": {
    "prompt": "A 2D flat art style campsite poster with the text “Kie AI Seedream 3.0 API”. The scene includes mountains, brown tents under a sunny sky, a rabbit, a deer, some birds, green grass, and a flowing river, all illustrated in clean vector art.",
    "image_size": "square_hd",
    "guidance_scale": 2.5
  }
}
